사용된 파이썬 버전은 3.12 입니다.
requirments.txt 에 있는 라이브러리를 'pip install -r requirements.txt' 명령어로 설치해 주세요
그리고 파이썬으로 'app.py' 파일을 실행해 주시면 됩니다.

실행할때 pwd 는 app.py 가 있는 폴더 내부여야 합니다.
리눅스가 아닌 환경에서는 FEATURE 를 정상적으로 파싱하지 못할 수 있습니다.

웹 서버가 성공적으로 실행되었다면
127.0.0.1:5000 에서 RTT 실제값, 예측값 그래프를 확인 할수 있으며
127.0.0.1:5000/monitor 에서 실시간으로 모델에 넘겨주는 features 를 확인 할 수 있습니다.

모델은 기본적으로 성능이 가장 우수했던 xgboost 모델을 사용합니다.
